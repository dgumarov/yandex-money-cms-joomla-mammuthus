///////////////////////////////////////
// Инструкция по установке модуля    //
// оплат через систему ЯНдекс.Деньги //
// для Joomla 1.5 + VirtueMart 1.1.9 //
///////////////////////////////////////

1. Для установки модуля оплат распакуйте архив дистрибутива во временный каталог.

2. Файлы платежного модуля нужно разместить в соответствующих каталогах движка Joomla:

2.1. Файлы модуля и конфига ps_yandex_money.php и ps_ yandex_money.cfg.php - в каталог 
	/administrator/components/com_virtuemart/classes/payment 

2.2. Файл взаимодействия yandex_money_notify.php - в каталог 
	/administrator/components/com_virtuemart 

3. В каталоге /images/yamoney нужно расположить пять иконок платежных опций: 
	ym_card.png, ym_terminal.png, ym_yandex_money.png,ym_mobile.png, ym_webmoney.png

Внимание! Проверьте, чтобы ps_yandex_money.cfg.php был доступен для записи, это 
необходимо только на время конфигурации модуля, в дальнейшем можно эти права на запись 
убрать.

Дальнейшая настройка модуля происзводиться из панели администрирования VirtueMart
